# Safety Call App

A mobile safety application built with React Native and Supabase to help users in uncomfortable or unsafe situations. Features include simulated phone calls, emergency location sharing, and direct emergency services integration.

## Features

- **Simulated Phone Calls** - Realistic fake calls with text-to-speech voice integration
- **Emergency Location Sharing** - One-tap SMS location sharing to emergency contacts
- **Emergency Services** - Direct 911 calling with automatic location sharing
- **Contact Management** - Manage emergency contacts with primary designation
- **Incident Logging** - Automatic logging of safety events with timestamps

## Tech Stack

**Frontend:** React Native • TypeScript • Expo • React Navigation

**Backend:** Supabase (PostgreSQL) • JWT Authentication • Row-Level Security

**Native Features:** Location Services • SMS • Text-to-Speech • Haptics
